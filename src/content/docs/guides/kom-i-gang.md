## Velkommen til webudvikling!

Dette undervisningsmateriale vil guide dig gennem de grundlæggende teknologier bag moderne websider: HTML, CSS og JavaScript.

### Hvad skal du lære?

<div class="card-grid">

**HTML (HyperText Markup Language)**

- Strukturen på en webside
- Semantiske elementer
- Formularer og input

**CSS (Cascading Style Sheets)**

- Styling og layout
- Flexbox og Grid

**JavaScript**

- Programmering grundlæggende

## Forudsætninger

For at komme i gang har du brug for:

- **En teksteditor** - f.eks. VS Code, Sublime Text eller Atom
- **En webbrowser** - Chrome, Firefox, Safari eller Edge
- **Nysgerrighed og tålmodighed** - læring tager tid!

## Din første HTML-side

Lad os starte med at oprette en simpel HTML-fil:

```html
<!DOCTYPE html>
<html lang="da">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Min første webside</title>
  </head>
  <body>
    <h1>Velkommen til webudvikling!</h1>
    <p>Dette er min første HTML-side.</p>
  </body>
</html>
```

### Sådan gør du:

1. Åbn din teksteditor
2. Opret en ny fil kaldet `index.html`
3. Kopier koden ovenfor ind i filen
4. Gem filen
5. Åbn filen i din webbrowser
