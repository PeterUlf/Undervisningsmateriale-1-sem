En oversigt over de vigtigste tastaturgenveje til effektiv arbejdsgang. Når der står ⌘ i Mac kolonnen, skal du på en PC typisk bruge Ctrl i stedet.
:::tip[For PC brugere]

⌘ => Ctrl
:::

## Generelle genveje (Markering af tekst)

### Navigation og markering med tastaturet

| Funktion                          | Mac                    | PC (hvis anderledes)      |
| --------------------------------- | ---------------------- | ------------------------- |
| Marker til venstre/højre (tegn)   | `Shift + ←/→`          |                           |
| Marker op/ned (linje)             | `Shift + ↑/↓`          |                           |
| Marker til start/slut af ord      | `Shift + Option + ←/→` | `Shift + Ctrl + ←/→`      |
| Marker til start/slut af linje    | `Shift + ⌘ + ←/→`      | `Shift + Home/End`        |
| Marker til start/slut af dokument | `Shift + ⌘ + ↑/↓`      | `Shift + Ctrl + Home/End` |
| Marker hele ord                   | Dobbeltklik eller træk |                           |
| Udvid markering til ord           | `⌘ + Shift + D`        | `Ctrl + D`                |

:::tip[**OBS:** Dobbeltklik virker ikke?]
Har ordet bindestreg (-) eller underscore (\_)?
Eksempler:

- `hello-world` vil kun markere `hello` eller `world` ved dobbeltklik
- `variable_name` vil kun markere `variable` eller `name` ved dobbeltklik
- I VS Code kan du bruge `Shift + Option + →` (Mac) eller `Shift + Ctrl + →` (PC) for at markere hele udtryk med bindestreg eller underscore
  :::

</Slide>

## Generelle genveje (Markering af filer)

### I Finder (Mac) eller File Explorer (Windows)

| Funktion                       | Mac                       | PC (hvis anderledes) |
| ------------------------------ | ------------------------- | -------------------- |
| Marker alle filer              | `⌘ + A`                   |                      |
| Naviger mellem filer           | `↑/↓` eller `←/→`         |                      |
| Udvid markering med piletaster | `Shift + ↑/↓` eller `←/→` |                      |
| Åbn markeret fil               | `⌘ + O` eller `Enter`     | `Enter`              |

:::tip[Kombiner SHIFT og ⌘]
**Marker flere spredte filer:** Hold `⌘` (Mac) eller `Ctrl` (PC) nede og klik på hver fil du vil markere.

**Marker en række filer:** Klik på den første fil, hold `Shift` nede, og klik på den sidste fil. Alle filer imellem bliver markeret.

**Kombination:** Du kan kombinere de to teknikker - marker først en række med `Shift + klik`, og tilføj derefter enkelte filer med `⌘/Ctrl + klik`.
:::

</Slide>

## VS Code - Multi-cursor editing

### Markering og cursors at redigere på flere linier samtidigt.

| Funktion                | Mac                     | PC (hvis anderledes) |
| ----------------------- | ----------------------- | -------------------- |
| Tilføj cursor over      | `⌘ + Option + ↑`        | `Ctrl + Alt + ↑`     |
| Tilføj cursor under     | `⌘ + Option + ↓`        | `Ctrl + Alt + ↓`     |
| Marker næste forekomst  | `⌘ + D`                 |                      |
| Marker alle forekomster | `⌘ + Shift + L`         |                      |
| Tilføj cursor ved klik  | `Option + klik`         | `Alt + klik`         |
| Column selection        | `Shift + Option + træk` | `Shift + Alt + træk` |

:::tip[Tip til multi-cursor]
For at redigere samme tekst på flere næsten ens linier:

1. Marker den del af teksten du vil ændre på første linje
2. Tryk `⌘ + D` (Mac) eller `Ctrl + D` (PC) flere gange for at markere samme tekst på de næste linier
3. Begynd at skrive - ændringen sker på alle markerede steder samtidigt
   :::
