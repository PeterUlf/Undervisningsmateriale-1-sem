## Scroll Snap

<a
  href="https://codepen.io/PeterUlf/pen/myPGNOR"
  target="_blank"
  style="display: inline-flex; align-items: center; gap: 0.5rem; padding: 0.5rem 1rem; background-color: var(--sl-color-accent-low); color: var(--sl-color-white); border: 1px solid var(--sl-color-accent); border-radius: 0.5rem; text-decoration: none; font-weight: 600; font-size: var(--sl-text-sm); transition: background-color 0.2s; white-space: nowrap;"
>
  Se Scroll Snap eksempel på CodePen →
</a>

</div>

Lav en oplevelse der minder om TikTok eller præsentations-slides, hvor scroll "låser" sig fast til bestemte sektioner.

### scroll-snap-type

Sættes på containeren og bestemmer **hvordan** og **hvor strengt** scroll-snap skal fungere.

**Syntax:** `scroll-snap-type: [retning] [strenghed]`

**Retning:**

- `y` = lodret scroll (op/ned)
- `x` = vandret scroll (venstre/højre)
- `both` = begge retninger

**Strenghed:**

- `mandatory` = SKAL altid snappe til et snap-punkt (streng)
- `proximity` = snapper kun hvis du er tæt på et snap-punkt (blød)

### scroll-snap-align

Sættes på child-elementerne og bestemmer **hvor** på elementet snap-punktet skal være.

**Værdier:**

- `start` = toppen/venstre kant
- `center` = midten af elementet
- `end` = bunden/højre kant

```css
.container {
  scroll-snap-type: y mandatory; /* Lodret scroll, skal altid snappe */
  overflow-y: scroll;
  height: 100vh;
}

.section {
  scroll-snap-align: start; /* Snap til toppen af sektionen */
  height: 100vh;
}
```

## Scroll-driven Animations

  <a
    href="https://codepen.io/PeterUlf/pen/gbrdVgW"
    target="_blank"
    style="display: inline-flex; align-items: center; gap: 0.5rem; padding: 0.5rem 1rem; background-color: var(--sl-color-accent-low); color: var(--sl-color-white); border: 1px solid var(--sl-color-accent); border-radius: 0.5rem; text-decoration: none; font-weight: 600; font-size: var(--sl-text-sm); transition: background-color 0.2s; white-space: nowrap;"
  >
    Progress Bar - CodePen →
  </a>
</div>

Med moderne CSS kan vi animere elementer baseret på hvor langt man har scrollet, helt uden JavaScript.

_Bemærk: Dette er en nyere teknologi, men understøttes i Chrome, Edge og nyere browsere._

### Eksempel: Progress Bar

En bar i toppen der vokser mens man scroller ned ad siden.

```css
@keyframes grow-progress {
  from {
    scale: 0 1;
  }
  to {
    scale: 1 1;
  }
}

.progress-bar {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 5px;
  background: red;
  transform-origin: 0 50%;

  /* Kobl animationen til scroll i stedet for tid */
  animation: grow-progress auto linear;
  animation-timeline: scroll();
}
```

```css
.box {
  animation: appear-and-rotate linear both;
  animation-timeline: view();
  animation-range: entry 0% cover 50%;

@keyframes appear-and-rotate {
  from {
    opacity: 0;
    translate: -100px 0;
    rotate: -180deg;

    opacity: 1;
```

En klassisk effekt hvor baggrundsbilledet bevæger sig i et andet tempo end indholdet, hvilket skaber en følelse af dybde.

<ParallaxHero />

```css
/* Nøgle-CSS for Parallax */
.parallax-bg {
  animation: parallax-move linear both;
  animation-timeline: view();
  animation-range: entry 0% exit 100%;
}

@keyframes parallax-move {
  from {
    translate: 0 -15%;
  }
  to {
    translate: 0 0%;
  }
}
```

## Vandret Scroll Sektion

<a
  href="https://codepen.io/PeterUlf/pen/vEGzoqe"
  target="_blank"
  style="display: inline-flex; align-items: center; gap: 0.5rem; padding: 0.5rem 1rem; background-color: var(--sl-color-accent-low); color: var(--sl-color-white); border: 1px solid var(--sl-color-accent); border-radius: 0.5rem; text-decoration: none; font-weight: 600; font-size: var(--sl-text-sm); transition: background-color 0.2s; white-space: nowrap;"
>
  Se Vandret Scroll eksempel på CodePen →
</a>

</div>

En avanceret teknik hvor siden ser ud til at scrolle vandret, mens man scroller nedad. Dette gøres ved at kombinere en meget høj container med `position: sticky`.

<HorizontalScroll />

```css
/* Nøgle-CSS for Vandret Scroll */
.container {
  height: 300vh; /* Definerer hvor længe vi scroller */
  view-timeline: --h-scroll block;
}

.track {
  position: sticky;
  top: 20vh;
  animation: move-horizontal linear both;
  animation-timeline: --h-scroll;
}

@keyframes move-horizontal {
  to {
    translate: -100% 0;
  }
}
```
