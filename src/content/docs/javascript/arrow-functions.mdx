## Hvad er en arrow function?

En arrow function er en kortere måde at skrive funktioner på.

```javascript
// Traditionel funktion
function add(a, b) {
  return a + b;
}

// Arrow function
const addArrow = (a, b) => a + b;

// Sådan ser du resultatet i konsollen
console.log(add(2, 3)); // 5
console.log(addArrow(2, 3)); // 5
```

Begge versioner gør det samme.

## Grundsyntaks

```javascript
// Ingen parametre
const sayHi = () => "Hej";

// Én parameter
const double = (x) => x * 2;

// Flere parametre
const multiply = (a, b) => a * b;
```

## Kort og lang arrow function

```javascript
// Kort version (implicit return)
const square = (x) => x * x;

// Lang version (når du har flere linjer)
const squareLong = (x) => {
  const result = x * x;
  return result;
};
```

Brug den lange version, når funktionen kræver flere trin.

## Eksempel med arrays

```javascript
const numbers = [1, 2, 3, 4];

// Loop gennem arrayet og log fordoblede tal
numbers.forEach((number) => {
  console.log(number * 2);
});

// Output i konsollen:
// 2
// 4
// 6
// 8
```

`forEach` betyder: "kør denne kode én gang for hvert element i arrayet".

`number` er parameteren i arrow-funktionen.  
Det er bare et variabelnavn for "det tal vi er på lige nu".

Når koden kører, sker det sådan her:

- 1. omgang: `number` er `1` -> logger `2`
- 2. omgang: `number` er `2` -> logger `4`
- 3. omgang: `number` er `3` -> logger `6`
- 4. omgang: `number` er `4` -> logger `8`

Arrow functions bruges ofte i `forEach`, `map` og `filter`.

## Øvelse 1: Skriv om til arrow

Skriv disse to om til arrow functions:

```javascript
function greet(name) {
  return "Hej " + name;
}

function minus(a, b) {
  return a - b;
}
```

Log derefter resultaterne i konsollen.

:::tip[Hint]
Omskriv begge funktioner til arrow functions.
Test derefter at de virker ved at køre dem og logge resultaterne i konsollen.

## Hvad gør `push`?

`push` tilføjer en værdi bagerst i et array.

```javascript
const values = [];

values.push(10); // [10]
values.push(20); // [10, 20]

console.log(values); // [10, 20]
```

I øvelse 3 bruger du `push` til at lægge hver ny pris med moms ind i `withMoms`.

## Hvad gør `.map`?

`.map` gennemgår et array og returnerer et **nyt** array.

Det betyder:

1. Original-arrayet ændres ikke
2. Hvert element bliver transformeret
3. Resultatet samles i et nyt array

```javascript
const prices = [100, 250, 400];

const withMoms = prices.map((price) => price * 1.25);

console.log(prices); // [100, 250, 400]
console.log(withMoms); // [125, 312.5, 500]
```

```javascript
const pricesExMoms = pricesInclMoms.map((price) => ...);
```
