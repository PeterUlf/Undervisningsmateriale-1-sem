## Hvad er Arrow Functions?

**Arrow functions** er en kortere syntaks til at skrive funktioner i JavaScript. De blev introduceret i ES6 (2015).

<div class="columns-2">

<div>

### Traditionel funktion

```javascript
function greet(name) {
  return "Hej " + name;
}

console.log(greet("Maria")); // "Hej Maria"
```

</div>

<div>

### Arrow function

```javascript
const greet = (name) => {
  return "Hej " + name;
};

console.log(greet("Maria")); // "Hej Maria"
```

</div>

</div>

**Fordele:**

- Kortere syntaks
- Lettere at læse
- Nyttig ved callbacks og loops

## Arrow Function Syntaks

### Grundlæggende syntaks

```javascript
// Traditionel funktion
function add(a, b) {
  return a + b;
}

// Arrow function
const add = (a, b) => {
  return a + b;
};
```

### Implicit return (én linje)

Hvis funktionen kun består af én linje, kan du udelade `{}` og `return`:

```javascript
// Kort version - returnerer automatisk
const add = (a, b) => a + b;

console.log(add(5, 3)); // 8
```

### Én parameter - skip parenteserne

```javascript
// Med parenteser
const double = (x) => x * 2;

// Uden parenteser (kun ved ÉN parameter)
const double = (x) => x * 2;

console.log(double(5)); // 10
```

## Forskellige eksempler

```javascript
// Ingen parametre - brug tomme parenteser
const sayHello = () => "Hello!";
console.log(sayHello()); // "Hello!"

// Én parameter - parenteser valgfri
const square = (x) => x * x;
console.log(square(4)); // 16

// Flere parametre - kræver parenteser
const multiply = (a, b) => a * b;
console.log(multiply(3, 4)); // 12

// Multi-line body - kræver brackets og return
const calculatePrice = (price, tax) => {
  const total = price + price * tax;
  return total;
};
console.log(calculatePrice(100, 0.25)); // 125
```

## Arrow Functions med Arrays

Arrow functions er **perfekte** til array metoder som `.map()`, `.filter()` og `.forEach()`:

### forEach

```javascript
const products = ["Laptop", "Mouse", "Keyboard"];

// Traditionel funktion
products.forEach(function (product) {
  console.log(product);
});

// Arrow function (kortere!)
products.forEach((product) => {
  console.log(product);
});

// Endnu kortere hvis kun én linje
products.forEach((product) => console.log(product));
```

## .map() med Arrow Functions

`.map()` skaber et nyt array ved at transformere hvert element:

```javascript
const prices = [100, 200, 300];

// Traditionel funktion
const withTax = prices.map(function (price) {
  return price * 1.25;
});

// Arrow function
const withTax = prices.map((price) => price * 1.25);

console.log(withTax); // [125, 250, 375]
```

### Mere komplekst eksempel

```javascript
const products = [
  { name: "Laptop", price: 8999 },
  { name: "Mouse", price: 299 },
  { name: "Keyboard", price: 599 },
];

// Hent kun navnene
const names = products.map((product) => product.name);
console.log(names); // ["Laptop", "Mouse", "Keyboard"]

// Tilføj moms til priser
const withTax = products.map((product) => ({
  name: product.name,
  price: product.price * 1.25,
}));
```

:::tip[Bemærk]
Ved arrow functions der returnerer objekter, skal du wrappe objektet i `()`: `=> ({...})`
:::

## .filter() med Arrow Functions

`.filter()` skaber et nyt array med kun elementer der opfylder en betingelse:

```javascript
const products = [
  { name: "Laptop", price: 8999, inStock: true },
  { name: "Mouse", price: 299, inStock: false },
  { name: "Keyboard", price: 599, inStock: true },
];

// Find produkter under 1000 kr
const affordable = products.filter((product) => product.price < 1000);
console.log(affordable);
// [{name: "Mouse", ...}, {name: "Keyboard", ...}]

// Find produkter på lager
const available = products.filter((product) => product.inStock);
console.log(available);
// [{name: "Laptop", ...}, {name: "Keyboard", ...}]

// Kombiner flere betingelser
const cheapAndAvailable = products.filter(
  (product) => product.price < 1000 && product.inStock,
);
```

## Praktisk eksempel: Produktliste

```javascript
const products = [
  { id: 1, name: "Laptop", price: 8999, category: "Electronics" },
  { id: 2, name: "Mouse", price: 299, category: "Electronics" },
  { id: 3, name: "Desk", price: 2999, category: "Furniture" },
  { id: 4, name: "Chair", price: 1999, category: "Furniture" },
];

// Find elektronik under 1000 kr
const cheapElectronics = products
  .filter((p) => p.category === "Electronics")
  .filter((p) => p.price < 1000);

console.log(cheapElectronics); // [{id: 2, name: "Mouse", ...}]

// Få array af kun navne for alle møbler
const furnitureNames = products
  .filter((p) => p.category === "Furniture")
  .map((p) => p.name);

console.log(furnitureNames); // ["Desk", "Chair"]

// Tilføj moms til alle priser
const withTax = products.map((p) => ({
  ...p, // Spread operator - kopier alle properties
  priceWithTax: p.price * 1.25,
}));
```

## Almindelige mønstre

### Event handlers

```javascript
// Traditionel
button.addEventListener("click", function () {
  console.log("Clicked!");
});

// Arrow function
button.addEventListener("click", () => {
  console.log("Clicked!");
});
```

### setTimeout / setInterval

```javascript
// Vent 2 sekunder, så udfør funktion
setTimeout(() => {
  console.log("Dette vises efter 2 sekunder");
}, 2000);

// Gentag hvert sekund
setInterval(() => {
  console.log("Tick...");
}, 1000);
```

## Opgaver

### Opgave 1: Konverter til arrow functions

Konverter disse traditionelle funktioner til arrow functions:

```javascript
function double(x) {
  return x * 2;
}

function greet(name) {
  return "Hej " + name;
}

function calculateTotal(price, quantity) {
  return price * quantity;
}
```

### Opgave 2: Array manipulation

Givet dette array:

```javascript
const products = [
  { name: "Phone", price: 5999, inStock: true },
  { name: "Tablet", price: 3999, inStock: false },
  { name: "Laptop", price: 8999, inStock: true },
  { name: "Mouse", price: 299, inStock: true },
];
```

Brug arrow functions til at:

1. Få et array med kun navnene
2. Find produkter under 4000 kr
3. Find produkter der er på lager
4. Tilføj 25% moms til alle priser
