## Intro: Sådan bruger du opgaverne

API link:

[https://kea-alt-del.dk/t7/api](https://kea-alt-del.dk/t7/api)

Sådan gør du i hver opgave:

1. Kopier JavaScript-koden ind i CodePen (JS-feltet).
2. Ret **kun** delen med prikker i `const url`.
3. Åbn Console og tjek om output matcher opgaven.

Du skal altså kun skrive den sidste del af URL'en (path + eventuelle query-parametre).

## Opgave 1

Tjek i console:

- `Antal: 10`

```javascript

fetch(url)
  .then((res) => res.json())
  .then((data) => {
    console.log("Mine data i opgave 1 er: ", data);
```
