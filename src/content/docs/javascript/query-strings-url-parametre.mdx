## Intro

API link:

[https://kea-alt-del.dk/t7/api](https://kea-alt-del.dk/t7/api)

På næste slide får du én stor kodeboks med alle 20 opgaver.

Sådan bruger du den i CodePen:

1. Kopier hele boksen.
2. Udfyld `urlParametre` i den opgave du vil løse.
3. Kald funktionen i bunden, fx `opgave1();`.
4. Løs én opgave ad gangen.

  /*Opgave 1 Hent produktlisten med standardindstillingerne (første 10 produkter).*/
  const urlGrund = "https://kea-alt-del.dk/t7/api/";
  const urlParametre = "..."; // skriv eventuelle query-parametre her

  fetch(urlFuld)
    .then((res) => res.json())
    .then((data) => {
      console.log("Opgave 1 data er: ", data);
    });
}
