## Hvad er et objekt?

Et objekt samler relaterede data i **key/value-par**.

```javascript
// Vi laver et objekt med tre key/value-par
const student = {
  name: "Asta", // key = name, value = tekst
  age: 21, // key = age, value = tal
  active: true, // key = active, value = boolean
};
```

Her er `name`, `age` og `active` keys. Værdierne er henholdsvis en tekst, et tal og en boolean.

## Læs værdier i et objekt

Du kan læse værdier med **dot notation** eller **bracket notation**.

```javascript
const student = {
  name: "Asta",
  age: 21,
};

// Dot notation: når du kender key-navnet på forhånd
console.log(student.name); // Asta

// Bracket notation: når key-navnet fx ligger i en variabel
console.log(student["age"]); // 21
```

Brug oftest dot notation. Bracket notation er nyttig, når key-navnet ligger i en variabel.

## Opdater og tilføj data

Objekter kan ændres efter de er oprettet.

```javascript
const student = {
  name: "Asta",
  age: 21,
};

student.age = 22; // opdaterer eksisterende key
student.city = "København"; // tilføjer ny key

console.log(student); // { name: "Asta", age: 22, city: "København" }
```

## Objekter kan være indlejrede

Et objekt kan indeholde andre objekter og arrays.

```javascript
const gymClass = {
  subject: "Samfundsfag B", // almindelig key
  teacher: {
    // indlejret objekt
    name: "Mette",
    classroom: "2.14",
  },
  modules: ["Demokrati", "Medier", "Økonomi"], // array inde i objekt
};

console.log(gymClass.teacher.name); // Mette (objekt -> key)
console.log(gymClass.modules[0]); // Demokrati (array -> index 0)
```

## Hvad er JSON?

JSON står for **JavaScript Object Notation**.

JSON er et **tekstformat** til at gemme og sende data, fx mellem browser og server.

```json
{
  "name": "Asta",
  "age": 21,
  "active": true
}
```

## Objekt vs JSON

De ligner hinanden, men er ikke det samme.

```javascript
// JavaScript objekt
const student = {
  name: "Asta",
  age: 21,
};
```

```json
// JSON (tekstformat)
{
  "name": "Asta",
  "age": 21
}
```

Vigtig forskel: I JSON skal keys stå i dobbelte anførselstegn.

## Øvelse 1: Lav et objekt

Lav et objekt `product` med disse keys:

1. `name`
2. `price`
3. `inStock`

Log alle tre værdier i konsollen.

:::tip[Hint: Hvad skal i `{}` og i anførselstegn?]
I objektet (mellem `{ ... }`) skriver du key/value-par.

- `name`, `price` og `inStock` er keys
- Tekst (string) skal i anførselstegn: `"T-shirt"`
- Tal og true/false skal ikke i anførselstegn: `199`, `true`

Brug derefter `console.log(...)` til at skrive noget i browserens Console.

```javascript
console.log(product.name);
console.log(product.price);
console.log(product.inStock);
```

:::

Start med:

```javascript
const user = {
  username: "mikkel01",

```

1. Opdater `points` til 25
2. Tilføj key `level` med værdien `"beginner"`
3. Log hele objektet

:::tip[Hint: Opdater og tilføj key]
Du bruger samme syntaks begge gange: `objekt.key = værdi`.

```javascript
user._____ = _____; // opdater points
user._____ = "_____"; // tilføj level
console.log(_____); // log hele objektet
```

## Øvelse 3: Indsæt produkter med `forEach`

Du har denne HTML:

```html
<section id="productList"></section>
```

og disse data i JavaScript:

```javascript
const products = [
  { name: "Tuborg Classic", price: 14, inStock: true },
  { name: "Faxe Kondi", price: 12, inStock: false },
  { name: "Cocio", price: 15, inStock: true },
];
```

Opgave:

1. Find `#productList` med `querySelector`
2. Brug `forEach` til at gennemgå `products`
3. Indsæt ét `<article>` pr. produkt med `name`, `price` og `inStock`
4. Brug `${...}` til at indsætte værdierne

:::tip[Hint]
Start med at lave en variabel til containeren, og loop derefter med `forEach`.
Husk backticks `` `...` `` omkring HTML-strengen.

```javascript
const list = document.querySelector("#productList");
products.forEach((product) => {
  // byg ét article-element pr. produkt
});
```

Brug kun `forEach` i denne øvelse.

:::

## Hvordan ser du et array af objekter i JSON?

Kig efter formen:

- JSON starter med `[` og slutter med `]` -> så er det et array
- Hvert element i arrayet starter med `{` og slutter med `}` -> så er elementet et objekt

Eksempel:

```json
[
  { "name": "Beanie", "price": 149 },
  { "name": "Socks", "price": 79 }
]
```

Hurtigt tjek i JavaScript:

```javascript
// Her er JSON-teksten manuelt lavet om til JavaScript-data
const data = [
  { name: "Beanie", price: 149 },
  { name: "Socks", price: 79 },
];

console.log(Array.isArray(data)); // true
console.log(data[0]); // første objekt i arrayet
console.log(data[0].name); // key fra første objekt
```
