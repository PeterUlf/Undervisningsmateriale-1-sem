## JavaScript Objects

Et **object** er en samling af relaterede data og funktionalitet. Objekter består af **key-value pairs** (nøgle-værdi par).

```javascript
const product = {
  name: "Laptop",
  price: 8999,
  inStock: true,
  brand: "Dell",
};
```

Her er `name`, `price`, `inStock` og `brand` **keys** (nøgler), og deres tilsvarende værdier er strengen "Laptop", tallet 8999, osv.

## Tilgå objektets properties

Du kan tilgå værdierne i et objekt på to måder:

<div class="columns-2">

<div>

### Dot notation

```javascript
const product = {
  name: "Laptop",
  price: 8999,
};

console.log(product.name); // "Laptop"
console.log(product.price); // 8999
```

**Mest almindelig** - brug når du kender key-navnet.

</div>

<div>

### Bracket notation

```javascript
const product = {
  name: "Laptop",
  price: 8999,
};

console.log(product["name"]); // "Laptop"
console.log(product["price"]); // 8999
```

**Nyttigt når** key-navnet er i en variabel eller indeholder specialtegn.

</div>

</div>

</Slide>

## Nested Objects

Objekter kan indeholde andre objekter:

```javascript
const product = {
  name: "Laptop",
  price: 8999,
  specs: {
    ram: "16GB",
    storage: "512GB SSD",
    screen: "15.6 tommer",
  },
  manufacturer: {
    name: "Dell",
    country: "USA",
  },
};

// Tilgå nested properties
console.log(product.specs.ram); // "16GB"
console.log(product.manufacturer.name); // "Dell"
```

## Hvad er JSON?

**JSON** står for **JavaScript Object Notation**. Det er et tekstformat til at gemme og sende data.

### JSON ser næsten ud som JavaScript objekter:

<div class="columns-2">

<div>

**JavaScript Object:**

```javascript
const product = {
  name: "Laptop",
  price: 8999,
  inStock: true,
};
```

</div>

<div>

**JSON:**

```json
{
  "name": "Laptop",
  "price": 8999,
  "inStock": true
}
```

</div>

</div>

### Forskelle:

- JSON keys skal være i **anførselstegn** (`"name"`)
- JSON er **ren tekst** (string)
- Kan sendes mellem server og browser

</Slide>
<div class="page-break"></div>
<Slide>

## Konverter mellem JSON og JavaScript

### JSON til JavaScript (Parse)

```javascript
// JSON som tekst-string
const jsonString = '{"name":"Laptop","price":8999}';

// Konverter til JavaScript object
const product = JSON.parse(jsonString);

console.log(product.name); // "Laptop"
console.log(product.price); // 8999
```

### JavaScript til JSON (Stringify)

```javascript
// JavaScript object
const product = {
  name: "Laptop",
  price: 8999,
};

// Konverter til JSON string
const jsonString = JSON.stringify(product);

console.log(jsonString); // '{"name":"Laptop","price":8999}'
```

## Arrays af Objects

Ofte får du en liste (array) af objekter fra en API:

```javascript
const products = [
  {
    id: 1,
    name: "Laptop",
    price: 8999,
  },
  {
    id: 2,
    name: "Mouse",
    price: 299,
  },
  {
    id: 3,
    name: "Keyboard",
    price: 599,
  },
];

// Loop gennem produkter
products.forEach((product) => {
  console.log(product.name + ": " + product.price + " kr");
});
```

## Praktisk eksempel

```javascript
// Simulerer data fra en API
const apiResponse = `{
  "products": [
    {"id": 1, "name": "Laptop", "price": 8999, "image": "laptop.jpg"},
    {"id": 2, "name": "Mouse", "price": 299, "image": "mouse.jpg"},
    {"id": 3, "name": "Keyboard", "price": 599, "image": "keyboard.jpg"}
  ]
}`;

// Parse JSON til JavaScript
const data = JSON.parse(apiResponse);

// Tilgå produkter
data.products.forEach((product) => {
  console.log(`${product.name} koster ${product.price} kr`);
});

// Output:
// Laptop koster 8999 kr
// Mouse koster 299 kr
// Keyboard koster 599 kr
```
