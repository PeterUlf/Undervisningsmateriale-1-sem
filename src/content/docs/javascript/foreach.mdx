## Grundidéen i `forEach`

`forEach` kører en funktion én gang for hvert element i et array.

```javascript
const numbers = [10, 20, 30];

numbers.forEach((number) => {
  console.log(number);
});
```

Output:

- 10
- 20
- 30

## Eksempel med objekter

`forEach` bruges ofte på arrays af objekter:

```javascript
const products = [
  { name: "Tuborg Classic", price: 14 },
  { name: "Faxe Kondi", price: 12 },
  { name: "Cocio", price: 15 },
];

products.forEach((product) => {
  console.log(product.name, product.price);
});
```

Her er `products` arrayet, og `product` er ét objekt ad gangen.

## Brug `forEach` til HTML

### HTML

```html
<section id="productList"></section>
```

### JavaScript

```javascript
const products = [
  { name: "Tuborg Classic", price: 14 },
  { name: "Faxe Kondi", price: 12 },
  { name: "Cocio", price: 15 },
];

const list = document.querySelector("#productList");

list.innerHTML = "";

products.forEach((product) => {
  list.innerHTML += `
    <article class="productCard">
      <h3>${product.name}</h3>
      <p>Pris: ${product.price}</p>
    </article>
  `;
});
```

Dette mønster er det samme, du bruger i `fetch-productlist`, når data kommer fra API.

## Typiske fejl

1. Du bruger `=` i stedet for `+=` i `innerHTML`, så kun sidste element vises.
2. Du prøver at bruge `forEach` på noget, der ikke er et array.
3. Du forventer et nyt array som resultat.

`forEach` returnerer ikke et nyt array.  
Hvis du vil lave et nyt array, så brug `map`.

## Øvelse 1: Navn og index

Givet:

```javascript
const students = ["Asta", "Mikkel", "Noah", "Ida"];
```

Brug `forEach` til at logge:

- index
- navn

Eksempel på output:

`0: Asta`

:::tip[Hint]
Brug to parametre i callbacken: `(student, index) => { ... }`.
:::
