## Forklaring til øvelse 4

I øvelse 4 bruger du `filter`.

`filter` gennemgår et array og returnerer kun de elementer, der opfylder en betingelse.

```javascript

  { name: "Talk A", freeSeats: 0 },
  { name: "Talk B", freeSeats: 3 },
```

## Forklaring til øvelse 5

I øvelse 5 bruger du `sort` til pris-rækkefølge (billigst -> dyrest).

Det vigtige er:

1. `a` og `b` er to produkter, som `sort` sammenligner ad gangen
2. `a.price - b.price` er sammenligningen
3. Negativt tal: `a` kommer før `b`
4. Positivt tal: `b` kommer før `a`
5. `[...]` laver en kopi, så original-arrayet ikke ændres

```javascript
const items = [
  { name: "A", price: 90 },
  { name: "B", price: 30 },
  { name: "C", price: 60 },
];

// sort sammenligner to elementer ad gangen: a og b
// her sorterer vi stigende pris (billigst først)
const sorted = [...items].sort((a, b) => a.price - b.price);

// Eksempel på en sammenligning:
// 90 - 30 = 60  -> positivt tal -> den billigste (30) placeres foran

console.log(items); // original: [90, 30, 60]
console.log(sorted);
// sorteret: [30, 60, 90]
```

## Forklaring til øvelse 6

I øvelse 6 bruger du `async/await` og fejl-check.

Flow:

1. `await fetch(...)`
2. Check `response.ok`
3. `await response.json()`
4. `catch` hvis noget fejler

```javascript
const endpoint = "https://kea-alt-del.dk/t7/api/products/2050";

async function getOne() {
  try {
    const response = await fetch(endpoint);
    if (!response.ok) {
      throw new Error("Request fejlede");
    }
    const data = await response.json();
    console.log(data.productdisplayname);
  } catch (error) {
    console.error(error.message);
  }
}

getOne();
```

## Forklaring til øvelse 7

I øvelse 7 kombinerer du flere metoder i rækkefølge:

1. `fetch` -> hent produktliste
2. `filter` -> vælg kun en del af data
3. `sort` -> sorter pris
4. `slice` -> begræns antal
5. `forEach` -> vis i HTML

Eksempel på kæde:

```javascript
const result = data
  .filter((item) => item.active)
  .sort((a, b) => a.price - b.price)
  .slice(0, 5);
```
